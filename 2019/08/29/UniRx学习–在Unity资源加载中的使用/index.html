UniRx学习–在Unity资源加载中的使用

发表于 2019-08-29
本文字数： 5.2k | 阅读时长 ≈ 5 分钟

简介
ReactiveX.io给Rx定义是一个使用可观察数据流进行异步编程的编程接口，RX结合了观察者模式、迭代器模式和函数式编程的精华，可以帮助我们很快的完成一些异步操作，以下资料可以帮助快速入门。

原文地址：http://changxianjie.com/unirx%e5%ad%a6%e4%b9%a0-%e5%9c%a8unity%e8%b5%84%e6%ba%90%e5%8a%a0%e8%bd%bd%e4%b8%ad%e7%9a%84%e4%bd%bf%e7%94%a8/

Rx官方文档的翻译
在函数响应式编程中作者以通俗易懂的详细的解释了什么是Rx和Rx的使用方式。
UniRx是Rx在Unity上实现，在实现Rx同时，封装了一些Unity的api，使我们可以快速的完成一些功能的实现，在最新的Rx版本中，作者实现了UniTask,可以使用C#原生的async和await进行异步操作(需要支持C#7.0版本，Unity2018.3或者在unity2018其他版本中使用incrementalcompiler package包)。

在Unity上的实现从UniRx的Readme中，我们可以学习到UniRx的基本介绍和基本使用方法，同时在插件的Examples目录下还有作者写好的几个例子。

补充例子UniRx在Unity资源加载中的使用
如果不使用UniRx从Resources目录下加载资源可能会使用如下的写法

IEnumerator的写法
private IEnumerator LoadPrefabFromRes(string assetName, Action loaded, bool isAsync = false) where T : UnityEngine.Object
{
var assetObj = default(T);
if (isAsync)
{
ResourceRequest req = Resources.LoadAsync(assetName);
yield return req;
assetObj = req.asset as T;
}
else
{
assetObj = Resources.Load(assetName);
}
loaded?.Invoke(assetObj);
}
C#Copy
这样的写法不算复杂，但是上面的方法只能通过回调函数的方式使用加载好的资源，并且只用使用继承MonoBehaviour的对象调用，如果需要AssetBundle加载资源，还需要写写类似上述方式的加载ab的方法，然后在回调方法中调用上述方法，这样会出现双重回调，代码将非常的冗杂和层次很深，对代码的阅读和调试非常的不利，非常的不灵活。

IObservable方式
public IObservable LoadPrefabFromResObservable(string assetName, bool isAsync = true) where T : UnityEngine.Object
{
if (isAsync)
return Resources.LoadAsync(assetName)
.AsAsyncOperationObservable()
.Select(resourceRequest => resourceRequest.asset)
.OfType<UnityEngine.Object, T>();
return Observable.Start(() => Resources.Load(assetName), Scheduler.MainThread);
}
C#Copy
上面的代码中我们使用了UniRx是链式方法，只用了一句代码实现了Resources资源加载，同时LoadPrefabFromResObservable返回的是一个IObservable对象，在具体使用资源之前仍然可以使用UniRx的其他操作符继续操作。

此方式已经比IEnumerator方式简单灵活了好多。

AsAsyncOperationObservable 将ResourceRequest转换为UniRx中可观察的对象返回IObservable
Select 将IObservable转变为IObservable<UnityEngine.Object>
OfType 将IObservable<UnityEngine.Object>转变为IObservable
Observable.Start 创建一个新的IObservable对象
Scheduler.MainThread 表示在主线程上执行
调用方式如下

LoadPrefabFromResObservable(“texture1”).Subscribe(tex => rawImage1.texture = tex);
C#Copy
Subscribe 注册可观察对象
接下为最简单方便的方式

UniTask(async,await)方式
async UniTask LoadPrefabFromResUniTask(string assetName, bool isAsync = true) where T : UnityEngine.Object
{
if (isAsync)
return await Resources.LoadAsync(assetName) as T;
return Resources.Load(assetName);
}
C#Copy
UniRx可以使用await关键字完成Unity中Resources.LoadAsync并且直接返回加载好的UnityEngine.Object对象。

UniTask是C#Task对象在Unity中的实现，在原生C#中Task相关便利操作也被带入了unity中。

除此之外，UniRx也实现了Observable的操作符的await等待操作，使得这种写法更加的灵活简单。

调用方式如下

async void Start()
{
var text3 = await LoadPrefabFromResUniTask(“texture2”);
rawImage3.texture = text3;
}
C#Copy
使用await调用IObservable

async void Start()
{
var text2 = await LoadPrefabFromResObservable(“texture1”);
rawImage2.texture = text2;
}
C#Copy
总结
可以多多阅读UniRx参考解锁更多用法。

UniRx真乃简化工作提交效率的神器是也！

完整代码
使用Unity版本2018.3b8

using System;
using System.Collections;
using System.Threading;
using UniRx.Async;
using UnityEngine;
using UnityEngine.UI;

#if UNITY_2018_3_OR_NEWER

#pragma warning disable CS0618

#endif

namespace UniRx.Examples
{
public class UniRxTest : MonoBehaviour
{
public RawImage rawImage1;
public RawImage rawImage2;
public RawImage rawImage3;

    async void Start()
    {
        LoadPrefabFromResObservable<Texture2D>("texture1").Subscribe(tex => rawImage1.texture = tex);

        await UniTask.Delay(TimeSpan.FromSeconds(0.5));
        var text2 = await LoadPrefabFromResObservable<Texture2D>("texture1");
        rawImage2.texture = text2;

        await UniTask.Delay(TimeSpan.FromSeconds(0.5));
        var text3 = await LoadPrefabFromResUniTask<Texture2D>("texture2");
        rawImage3.texture = text3;
    }

    async UniTask<T> LoadPrefabFromResUniTask<T>(string assetName, bool isAsync = true) where T : UnityEngine.Object
    {
        if (isAsync)
            return await Resources.LoadAsync<T>(assetName) as T;
        return Resources.Load<T>(assetName);
    }

    public IObservable<T> LoadPrefabFromResObservable<T>(string assetName, bool isAsync = true) where T : UnityEngine.Object
    {
        if (isAsync)
            return Resources.LoadAsync<T>(assetName)
                .AsAsyncOperationObservable()
                .Select(resourceRequest => resourceRequest.asset)
                .OfType<UnityEngine.Object, T>();
        return Observable.Start(() => Resources.Load<T>(assetName), Scheduler.MainThread);

    }

    private IEnumerator LoadPrefabFromRes<T>(string assetName, Action<T> loaded, bool isAsync = false) where T : UnityEngine.Object
    {
        var assetObj = default(T);
        if (isAsync)
        {
            ResourceRequest req = Resources.LoadAsync<T>(assetName);
            yield return req;
            assetObj = req.asset as T;
        }
        else
        {
            assetObj = Resources.Load<T>(assetName);
        }
        loaded?.Invoke(assetObj);
    }
}

}

#pragma warning restore CS0618

#endif
