思考并回答以下问题：

一、对变量进行动画
Public Vector3 myValue = new Vector3(0,0,0);
//对变量做一个动画（通过插值的方式去修改一个值的变化）
DoTween.To(()=>myValue,x=>myValue=x,new Vector3(10,10,10),2);
其中第一个参数和第二个参数都是委托，代码中的前两个参数都是使用了Lambda表达式，第一个参数表示的是将myValue值返回出来，第二个参数是将DoTween计算好的值赋值给myValue（与第一个参数结合来说，就是将DoTween计算好的值赋值给myValue然后将myValue返回出来）；第三个参数表示的是目标位置，即myValue最终是（10，10，10）；第四个参数表示的是整个变化过程所需的时间。

十、DoTweenPath路径编辑器的使用

1）快捷键
SHIFT+CTRL：在场景视图中点击即可在点击的地方添加新的路径点；
SHIFT+ALT：在场景视图中点击某个路径点即可删除该路径点，该插件会通过剩下的路径点重新生成新路径。
2）路径总长度
Path Length：显示路径的总长度。
3)动画选项
AutoPlay：在开始运行时自动播放该动画；
AutoKill：在动画播放完毕后自动销毁该动画；
Duration：播放完该动画所需的时间（SpeedBased为灰色时），动画的播放速度（SpeedBased为绿色时）；
Delay：延迟多少秒才开始播放动画；
Ease：这是一个枚举，可以理解为动画播放速率曲线，以下是其中几种曲线；

http://robertpenner.com/easing/easing_demo.html
Loops：动画循环次数，默认为1（0为1次，-1为无限循环）；
ID：动画ID，用于在物体有多个DOTWEEN动画时用于区别；
Update Type：动画放在哪个Update中执行（有4种选择：None、LateUpdate、FixedUpdate、Manual）。
4）路径动画选项
Path Type：路径类型，设置路径点的两端是直线还是弧度；
Close Path：让开始位置和最后一个路径点连接起来（默认是不连接）；
Local Movement：相对坐标移动，若该物体的相对坐标和世界坐标不一致，则勾选上；
Path Mode：路径模式，保持默认；
Lock Rotation：锁定旋转（可选X轴、Y轴、Z轴、W轴<所有轴>）
Orientation：方向，默认为None（物体移动时角度不变，接下来皆指物体移动时Z轴正方向指向位置），可选的有To Path（指向路径正方向）、Look At Transform（指向指定物体的坐标）、Look At Position（指向指定坐标）
5）路径编辑选项
Relative：相对，勾选时，在场景视图中移动物体，该物体的整个路径会跟着移动（若移动路径点不会有这个效果）；Color：设置路径辅助线的颜色；
Show Indexes：显示路径点的顺序索引；
Show WPs Lengths：显示路径点之间的路径长度（非总长度，是当前点与上一个路径点之间的路径的长度）；
Live Preview：实时显示，保持默认即可；
Handles Type/Mode：不多说，看图（Handle Size：路径点大小）。

6）事件
Events：生命周期函数，根据需要点绿相应的事件，即可添加事件。
7）额外
Reset Path：清空路径（即清空所有路径点）；
Drop To Floor：当有路径点处于某物体的某个面上方时，点击该按钮，路径点会减少Y轴坐标直至贴合某个物体的某个面上（Offset Y：设置路径点与那个面之间的距离，默认为0，即没有距离，也就是贴合）；
Waypoints：所有路径点，点开该选项，可以看到已经存在的所有路径点。可以在这精准的修改坐标位置，并且可以改变路径点顺序（Copy to clipboard：复制所有路径点坐标，可以粘贴到文本中，就可以获取路径点的坐标代码）
